ChatGPT-Analysis
This project focuses on analyzing customer reviews of ChatGPT. The objective is to perform a comprehensive sentiment analysis and feature extraction from textual review data to understand user satisfaction and pinpoint positive or critical feedback patterns. The analysis includes data preprocessing, sentiment scoring, keyword extraction, and visualizations that reveal insights into user experiences.

The analysis aims to understand:
1. The general sentiment users express about ChatGPT.
2. The strength (subjectivity) of those sentiments.
3. The specific features or aspects that users most frequently praise or criticize.

The dataset contains 4 fields:
1. Review ID: A unique identifier for every review.
2. Review: Comments given by the user.
3. Rating: Represents the levels of satisfaction on a scale of 0-5.
4. Review date: When the review was posted.

The key objectives of this problem are:
1. Sentiment Analysis: Identify positive, negative, and neutral user sentiments.
2. Issue Identification: Detect common problems driving negative reviews.
3. Time-Series Analysis: Track how sentiment evolves over time.
